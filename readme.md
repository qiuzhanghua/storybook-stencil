# Use Storybook and Stencil to build a component library

This is a starter project for building a component library with Stencil and Storybook.

## Getting Started
```bash
npm stencil init
cd storybook-stencil
pnpm install
pnpm dlx storybook init --type html
ncu -n
pnpm install
pnpm build
pnpm storybook
```
